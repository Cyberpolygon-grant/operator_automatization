========================================
ИНСТРУКЦИЯ ПО АВТОЗАПУСКУ
Автоматизация оператора ДБО
========================================

ВАЖНО: Все файлы должны находиться в:
C:\Program Files\operator_automatization-main\1\

БЫСТРАЯ УСТАНОВКА:
------------------
1. Скопируйте файлы в:
   C:\Program Files\operator_automatization-main\1\
   
   Необходимые файлы:
   - dbo_automation.py
   - start_automation.bat
   - start_automation_hidden.vbs

2. Запустите: install_autostart.bat
   (из папки C:\Program Files\operator_automatization-main\1\)

3. Готово! Скрипт будет запускаться СКРЫТО при входе в систему
   и автоматически перезапускаться при завершении
   Система НЕ будет спрашивать о доверии (используется ярлык)

ОСОБЕННОСТИ:
------------
✓ Python скрипт запускается при входе в систему
✓ Работает СКРЫТО (консоль не видна пользователю)
✓ Автоматически перезапускается при завершении (ошибка, закрытие и т.д.)
✓ Работает в фоновом режиме
✓ Все файлы в Program Files
✓ Нет запроса о доверии (используется ярлык вместо прямого VBS)

КАК ОСТАНОВИТЬ:
---------------
1. Запустите: uninstall_autostart.bat
   (из папки C:\Program Files\operator_automatization-main\1\)

2. Или вручную:
   - Откройте Диспетчер задач (Ctrl+Shift+Esc)
   - Найдите процесс cmd.exe (start_automation.bat)
   - Завершите процесс
   - Или удалите файл из папки автозагрузки:
     Win+R → shell:startup → удалите dbo_automation_startup.lnk

ПРОВЕРКА РАБОТЫ:
---------------
Откройте Диспетчер задач (Ctrl+Shift+Esc)
Найдите процессы:
- cmd.exe (start_automation.bat) - обертка с автоперезапуском
- python.exe (dbo_automation.py) - основной скрипт

Если процессы есть - скрипт работает!

ОСОБЕННОСТИ:
------------
✓ Обрабатываются только файлы, созданные ПОСЛЕ запуска ПК
✓ Старые файлы автоматически пропускаются
✓ Файлы удаляются через 10 минут после скачивания
✓ При завершении Python скрипта - автоматически перезапускается через 3 секунды
✓ Консоль полностью скрыта от пользователя
✓ Нет запроса о доверии при запуске

ФАЙЛЫ:
------
- dbo_automation.py - основной Python скрипт
- start_automation.bat - обертка с автоперезапуском
- start_automation_hidden.vbs - VBS скрипт для скрытого запуска
- install_autostart.bat - установка автозапуска
- uninstall_autostart.bat - удаление автозапуска
