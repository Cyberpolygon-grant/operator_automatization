========================================
ИНСТРУКЦИЯ ПО АВТОЗАПУСКУ
Автоматизация оператора ДБО
========================================

ВАЖНО: Все файлы должны находиться в:
C:\Program Files\operator_automatization-main\1\

БЫСТРАЯ УСТАНОВКА:
------------------
1. Скопируйте ВСЕ файлы из этой папки в:
   C:\Program Files\operator_automatization-main\1\
   
   Необходимые файлы:
   - dbo_automation.py
   - start_automation.bat
   - start_automation_hidden.bat
   - install_autostart_simple.bat
   - stop_automation.bat
   - uninstall_autostart.bat

2. Запустите: install_autostart_simple.bat
   (из папки C:\Program Files\operator_automatization-main\1\)

3. Готово! Скрипт будет запускаться СКРЫТО при входе в систему

ОСОБЕННОСТИ:
------------
✓ Скрипт работает СКРЫТО (окно не видно)
✓ При закрытии окна - автоматически перезапускается
✓ При ошибке - автоматически перезапускается
✓ Защищен от случайного завершения
✓ Поддержка UTF-8 (правильное отображение русских символов)
✓ Все файлы в Program Files

ФАЙЛЫ:
------
- start_automation_hidden.bat - скрипт скрытого запуска (без запроса о доверии)
- start_automation.bat - рабочий скрипт (запускается скрыто)
- install_autostart_simple.bat - установка автозапуска
- stop_automation.bat - остановка автоматизации
- uninstall_autostart.bat - удаление автозапуска

КАК ОСТАНОВИТЬ:
---------------
1. Запустите: stop_automation.bat
2. Или удалите файл из папки автозагрузки:
   - Win+R → shell:startup
   - Удалите: start_automation_hidden.bat

ПРОВЕРКА РАБОТЫ:
---------------
Откройте Диспетчер задач (Ctrl+Shift+Esc)
Найдите процессы:
- python.exe (dbo_automation.py)
- cmd.exe (start_automation.bat)

ОСОБЕННОСТИ:
------------
✓ Обрабатываются только файлы, созданные ПОСЛЕ запуска ПК
✓ Старые файлы автоматически пропускаются
✓ Нет запроса о доверии к файлу (используется bat вместо VBS)
✓ Файлы удаляются через 10 минут после скачивания

Если процессы есть - скрипт работает!

ЛОГИ:
-----
В скрытом режиме логи сохраняются в файл:
C:\Program Files\operator_automatization-main\1\automation.log
