========================================
ИНСТРУКЦИЯ ПО АВТОЗАПУСКУ
Автоматизация оператора ДБО
========================================

БЫСТРАЯ УСТАНОВКА:
------------------
1. Запустите: install_autostart_simple.bat
2. Готово! Скрипт будет запускаться СКРЫТО при входе в систему

ОСОБЕННОСТИ:
------------
✓ Скрипт работает СКРЫТО (окно не видно)
✓ При закрытии окна - автоматически перезапускается
✓ При ошибке - автоматически перезапускается
✓ Защищен от случайного завершения

ФАЙЛЫ:
------
- start_automation_hidden.vbs - основной скрипт (скрытый запуск)
- start_automation.bat - рабочий скрипт (запускается скрыто)
- install_autostart_simple.bat - установка автозапуска
- stop_automation.bat - остановка автоматизации

КАК ОСТАНОВИТЬ:
---------------
1. Запустите: stop_automation.bat
2. Или удалите файл из папки автозагрузки:
   - Win+R → shell:startup
   - Удалите: start_automation_hidden.vbs

ПРОВЕРКА РАБОТЫ:
---------------
Откройте Диспетчер задач (Ctrl+Shift+Esc)
Найдите процессы:
- python.exe (dbo_automation.py)
- wscript.exe (start_automation_hidden.vbs)

Если процессы есть - скрипт работает!
